# Stack Game with Unity-Tuanjie

Stack Wearable is a smartwatch hyper-casual game built with Unity China (Tuanjie Hub OpenHarmony build) and running on HarmonyOS smart wearable devices. Tap at the right time to stack moving blocks, build the highest tower you can, and enjoy quick, one-touch gameplay optimized for the watch screen.
